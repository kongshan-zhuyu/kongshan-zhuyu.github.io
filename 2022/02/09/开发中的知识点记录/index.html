VS Code

Ctrl+Shift+P：根据您当前的上下文访问所有可用命令

Ctrl+P：快速打开文件

Ctrl+Shift+M：快速跳转到项目中的错误和警告

Ctrl+K M：保留该文件类型的新语言模式

Ctrl+K Ctrl+T：从 VS Code 扩展市场安装更多主题

Ctrl+B：显示和隐藏状态栏

Ctrl+K Z：进入无干扰禅模式

Ctrl+\：拖放编辑器来创建新的编辑器组并在组之间移动编辑器

Ctrl+Shift+L：向当前选择的所有匹配项添加额外的光标

Ctrl+D：这只会选择您选择的下一个事件，以便您可以一个接一个地添加选择

Shift+Alt+Up或`Shift+Alt+Down：复制整行并将其粘贴到上方或下方的行中

option + ↑或`option + ↓：根据需要将整条线上下移动

Ctrl+K Ctrl+F：当前选择的源代码，Shift+Alt+F：整个文件格式

Ctrl+Shift+V：Markdown 文件见 Markdown 预览

Ctrl+K V：预览和编辑器将与您在任一视图中的滚动同步

Ctrl+Space：查看可用方法、参数提示、简短文档等
